# Netflix_Clone_Complete
It is a Fully functional Netflix clone, It looks and works the same as the Netflix site ,the users can Watch trailers ,choose the desired movie or series and can easily watch it here .

## Available Scripts

In the root of the project run:

### `npm install`

Also, navigate to the server folder and install modules there. You can do that by running:

### `cd server && npm install`

Afterwards, start the server from the server folder by running:

### `node index.js`

or 

### `nodemon index.js`

And to start the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

Go back to the project directory, you can run:

### `npm start`
